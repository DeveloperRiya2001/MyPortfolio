## 📌 Overview

This is my **personal portfolio website** — designed to give recruiters, hiring managers, and collaborators a fast, clear picture of who I am, what I've built, and what I can bring to a team.

Built entirely in **vanilla HTML, CSS, and JavaScript** — no frameworks, no build tools, no bloat. Just clean, semantic code that loads instantly and works everywhere.

## ✨ Features

- **Fully Responsive** — Mobile-first design, works flawlessly on all screen sizes
- **Smooth Scroll Navigation** — Sticky header with active-link highlighting
- **Scroll Reveal Animations** — Sections animate in as you scroll, powered by IntersectionObserver
- **Animated Skill Bars** — Progress bars trigger on viewport entry
- **Contact Form with Validation** — Client-side validation with success state
- **Zero Dependencies** — No npm, no webpack, no React — open `index.html` and go
- **SEO-Friendly** — Semantic HTML5, meta description, and clean heading hierarchy
- **Deploy-Ready** — Drop straight onto Netlify or Vercel

## ⚡ Getting Started

### Run Locally

```bash

# 2. Navigate into the folder

# 3. Open in your browser

# or just double-click index.html in your file explorer
```

## 🌐 Deployment

### Deploy to Netlify (recommended — 30 seconds)

1. Go to **[netlify.com/drop](https://netlify.com/drop)**
2. Drag and drop the `portfolio/` folder
3. Your site is live instantly — copy the URL

### Deploy to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy from the portfolio folder
vercel
```

### Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **`main` branch / root**
4. Your site will be live at `https://Riya-Kumari.github.io/portfolio`

## 🔧 Customisation

All content is in `index.html`. To update it for your own use:

- **Name / title / tagline** → Search for `Riya Kumari` and replace
- **Contact details** → Update `href="mailto:..."`, `href="tel:..."`, and LinkedIn/GitHub URLs
- **Projects** → Edit the `.project-card` sections in the Projects area
- **Skills / percentages** → Update `data-width="85"` values on `.skill-bar-fill` elements
- **Color palette** → Edit CSS variables in the `:root` block at the top of `<style>`
- **Resume download** → Replace `Riya_Resume.pdf` with your own file and update the `href`
